HeyHere (嘿，在这里~)
===========

## Installation(安装)

    pod install

## Usage(使用)

* Select the color, set the frequency of flashing, waving the phone,  let your friend find you

* 选取颜色，设置闪烁的频率，挥动手机，让朋友迅速找到你
